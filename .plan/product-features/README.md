# Product Features Plan

## Objective

Add the user-facing product workflows after the core chat foundation works.

## Scope

- Side panel onboarding.
- Settings/options page.
- Model visibility and favorites.
- Quick actions.
- Keyboard shortcut actions.
- Selected-text flow.
- Context-menu actions.
- AI suggestions stub and later implementation.

## Done When

- New users can complete setup inside the side panel.
- Users can manage provider/model/history settings.
- Quick actions send immediately and stream responses.
- Keyboard shortcuts reuse the same default quick action pipeline.
- Selected-text flow works through the same chat/action pipeline.
- AI suggestions setting exists, even if generated suggestions ship later.
